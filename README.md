

# 📘 HTML Practice Projects

This repository contains a collection of small projects and exercises I built while learning the basics of HTML.

## 🚀 About

These files are part of my learning journey, covering core HTML concepts like:

* Semantic structure
* Forms and inputs
* Tables
* Media elements (audio & video)
* Embedding content (iframes)
* Basic page layouts

## 📂 Projects Included

Some of the practice files in this repo:

* `form.html` – basic HTML form
* `survey_form.html` – survey form project
* `table.html` – working with tables
* `media_player.html` – audio/video usage
* `video_compilation.html` – embedded videos
* `bookstore.html` – simple layout example
* `travel_agency.html` – themed page design
* and more...

## 🧠 What I Learned

* Structuring web pages using HTML
* Using different HTML elements effectively
* Creating accessible and organized content
* Embedding media and external content

## 📌 Note

These are beginner-level projects created for practice, so the focus is on learning rather than perfect design or optimization.

## 🔧 How to Use

1. Clone the repository
2. Open any `.html` file in your browser
3. Explore and modify the code
